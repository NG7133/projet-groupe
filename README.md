PROJET SITE WEB - BOUTIQUE DE CHAUSSURES NIKE
     Description du projet
ce projet consiste a realiser un site web simple en HTML qui presente une boutique de chaussures. Le produit principal 
presente dans ce site est la chaussure Nike Air.

Le site permet de:
 -decouvrir la boutique
 -parcourir les categories de produits
 -consulter la fiche detaillee d'un produit
